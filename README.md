# Student Contact APP

[lsycool](https://github.com/lsycool/PictureDetect)

## Introduction

基于Face++在Android端实现的人脸识别分析软件，可自动识别所对应的人，并分析出目标年龄、种族等信息：

![Image text](https://github.com/lsycool/imgFolder/blob/master/splash.png)

## Usage

1. Star 本项目
2. Clone 到本地
3. 用Eclipse导入工程
4. 编译执行
5. APP安装到手机
6. Enjoy it!
